فحص معدل الطالبات عبر رقم الهوية، تم تحويل 23 ملف اكسل مباشرة إلى امتداد يسهل التعامل معه ثم ترتيب المعلومات في ملف واحد يشمل كل الطالبات، وإرفاقها في رابط الاستعلامبعد إجراء عملية حساب المعدل، علماً أن هناك تحدي وهو أن بعض الصفوف لديها مواد أكثر من صفوف أخرى ولكت تم أتمتة جميع المراحل، بطريقة منظمة في ملف واحد بامتداد csv .
